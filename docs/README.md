# 5G NR Layer 2 Protocol Stack Simulation
This project implements the User Plane (U-Plane) of the 5G NR Layer 2 stack, including PDCP, RLC, and MAC layers.

## 1. Installation (Requirement: User Manual)
To run this simulation, you must have `git` and `g++` installed. Follow these steps:

```bash
# Clone the repository
git clone [https://github.com/AlhassenSabeeh/5G_Layer2_Stack_Simulation.git](https://github.com/AlhassenSabeeh/5G_Layer2_Stack_Simulation.git)

# Navigate to the project directory
cd 5G_Layer2_Stack_Simulation
